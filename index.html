HTML

  • Head element contains information about the webpage
  • Title element contains the title of the webpage
  • Meta element contains information that
    • HTML Meta Tag
  • Body element defines the body of the document containing content visible to the user
    • Body element can be considered a semantic HTML element (describes the content and role on a webpage)
  • The header element contains the navigation menu/intro content
  • Div element creates a division in the page
  • Span element creates an inline division
  • Header elements like h1, h2, etc represent heading levels and can help in organizing a page
  • P element represents a paragraph or block of text (used to wrap text on webpage)
  • A ul and li element represents unordered lists and list items
  • IMG element contains info about images on a webpage (src attribute defines location of an image)
  • The br element creates a line of empty space/line break between content blocks
  • The a element stands for anchor creating links to same webpage or other webpages
    • The href element points to the URL

CSS

  • Cascading Style Sheets
  • Cascading Style Sheets (CSS) is used by devs to define how content is presented to users (colors, aesthetics, visual layout, etc)
  • Three ways to style a webpage with CSS
    • Inline CSS
      • When a developer wants to style an element directly in the HTML file, they use inline CSS
    • Internal CSS
      • When a developer wants to embed an ENTIRE stylesheet in the HTML file
    • External CSS
      • Developer rules kept in an external file
  • CSS Rules
    • Declarations: contain two important components: the CSS property and value
    • In our first CSS Sheet we have Margin and Padding
      • Margin: acts like a margine in a word processing document
        • Indicates how much space we want around the outside of an element
      • Padding: adds space around content INSIDE element. When you increase padding you are making the entire element bigger
      • img selector assigns property vlaues to any image element in the markup
        • Display property is assigned with the block value which assigns eash image to start on a new line
    • The Box Model
    • You can write attribute as the selector
      • There could be a scenario where we would like to add styles to certain elements but not all the elements
      • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
      • Then the class attributes can be assigned to the element in the HTML
  • We can combine two selectors into a single CSS rule with a comma allocwing the declarations to affect both selectors
  • CSS Card Attributes
    • The period preceding card designates this as a class selector. Any HTML elements assigned to the class will gain these property assignments from the CSS
    • Box-Shadow adds shadow effects around an elements frame

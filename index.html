HTML

  • Head element contains information about the webpage
  • Title element contains the title of the webpage
  • Meta element contains information that
  • Body element defines the body of the document containing content visible to the user
    • Body element can be considered a semantic HTML element (describes the content and role on a webpage)
  • The header element contains the navigation menu/intro content
  • Div element creates a division in the page
  • Span element creates an inline division
  • Header elements like h1, h2, etc represent heading levels and can help in organizing a page
  • P element represents a paragraph or block of text (used to wrap text on webpage)
  • A ul and li element represents unordered lists and list items
  • IMG element contains info about images on a webpage (src attribute defines location of an image)
  • The br element creates a line of empty space/line break between content blocks
  • The a element stands for anchor creating links to same webpage or other webpages
    • The href element points to the URL
